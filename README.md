# Farma Nuvem Popular

Sistema web para cadastro de pacientes, controle de receitas, registro de compras e auditoria de documentos da farmacia.

## Executar localmente

1. Instale as dependencias com `npm install`.
2. Copie `.env.example` para `.env` e informe as credenciais publicas do Supabase.
3. Execute `npm run dev`.

## Verificacao

- `npm run lint`
- `npm run build`

O esquema de referencia do banco esta em `supabase-schema.sql`.
